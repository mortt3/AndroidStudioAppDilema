Dilema es una aplicación de juegos sociales y de fiesta pensada para jugar con amigos. 
El objetivo de la app es facilitar la interacción entre los jugadores y crear un ambiente divertido mediante juegos clásicos y dinámicos de grupo, como Verdad o Reto y Tabú.

JUEGOS
  Actualmente la aplicación cuenta con algunos juegos implementados. Existen más juegos diseñados, 
  pero todavía no están integrados en la base de datos. Estos juegos se irán añadiendo progresivamente a 
  medida que avance el desarrollo del proyecto.

CATEGORÍAS
  Durante el juego, el usuario puede seleccionar el tipo de categoría según el nivel de intensidad del contenido:

    Soft
    Contenido ligero, adecuado para cualquier tipo de grupo.
    
    Party
    Contenido más atrevido, pensado para grupos con mayor confianza.
    
    Bofff
    Categoría pensada para contenido más intenso o explícito. Actualmente no se encuentra disponible ni implementada.

Los nombres y características de las categorías pueden cambiar durante el desarrollo.

ESTADO DEL PROYECTO

  Dilema se encuentra en una fase inicial de desarrollo. Se trata de un proyecto personal cuyo objetivo principal es el aprendizaje.

FUNCIONALIDADES FUTURAS

  Implementación de nuevos juegos
  
  Ampliación de la base de datos
  
  Mejora de las categorías y niveles de dificultad
  
  Optimización de la interfaz y usabilidad


  -------------------------------------------------------------------------  ENGLISH  -------------------------------------------------------------------------

  Dilema is a social and party games application designed to be played with friends. The goal of the app is to encourage interaction between players 
  and create a fun atmosphere through classic and group-based games such as Truth or Dare and Taboo.

  GAMES
    The application currently includes a limited number of implemented games. Additional games have been designed but are not yet 
    integrated into the database. These will be added gradually as development continues.
    
CATEGORIES
  During gameplay, users can select a category based on the desired level of intensity:

    Soft
    Light content suitable for any type of group.
    
    Hard
    More daring content, intended for groups with higher levels of trust.
    
    Extreme / NSFW
    Category intended for more intense or explicit content. This category is currently not implemented or available.

!!!!!!!! Dilema is currently in an early stage of development. This is a personal project focused on learning and progressively improving both technical aspects and user experience.
